Latest Translations (trunk)

The translations in this folder are the works-in-progress based on the latest PCSX2 trunk revisions, and may not match any specific PCSX2 revision.  The purpose of this folder is to provide revision control features for translation work being done on new and upcoming releases of PCSX2.

When PCSX2 is preparing for a non-beta release version (denoted by even numbering in the version digit, such as 0.9.8 or 1.0.0), relevant WIPs in this folder will be branched into an 0.9.8 folder, where translation work on a mostly stable non-moving target can be finalized.
